# Customer Churn Analysis and Prediction

## 📌 Project Overview
This project analyzes customer churn behavior using a telecom dataset. The goal is to identify key factors that influence churn and develop a predictive model to detect customers at risk of leaving.

## 📊 Dataset
The dataset includes customer demographics, service plans, usage behavior, and customer service interactions. The target variable is churn (True/False).

## 🔎 Key Steps
- Data cleaning and preparation  
- Exploratory data analysis (EDA)  
- Predictive modeling using Logistic Regression and Random Forest  
- Feature importance analysis  

## 📈 Key Findings
- Customers with frequent service calls showed higher churn risk  
- International plan users were more likely to churn  
- High daytime usage was strongly associated with churn  
- Voice mail plan users were less likely to churn  

## 🤖 Model Performance
The Random Forest model achieved strong performance and significantly outperformed logistic regression in detecting churned customers.

## 📌 Business Recommendations
- Improve customer service response strategies  
- Target high-risk customer segments  
- Offer personalized pricing plans  
- Promote value-added services
